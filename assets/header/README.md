# Header Images Folder

This folder contains images for the hero/header section of the website.

## Expected Files:
- Hero background images for the slider
- Hero thumbnail images

## Current Usage:
The website expects hero images at:
- `assets/hero/_DSC9585_1.jpg`
- `assets/hero/DSC_234_1.jpg`
- `assets/hero/DSC_2684_1.jpg`

Note: You may need to update the paths in `index.html` if you use this folder structure.
