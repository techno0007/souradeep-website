# Service Images Folder

This folder contains images for the service cards section.

## Expected Files:
- Wedding service card image
- Pre-Wedding service card image
- Baby Rice Ceremony service card image
- Birthday service card image

## Current Usage:
The website expects service images at:
- `assets/card/weeding/Wedding Thumbnail copy.jpg`
- `assets/card/preweeding/Pre Wedding Thumbnail copy.jpg`
- `assets/card/baby/24-09-03_12-28-03-838.jpg`
- `assets/card/birthday/Birthday Thumbnail.jpg`

Note: You may need to update the paths in `index.html` if you use this folder structure.
